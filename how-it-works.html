How Nameck Studio works

Nameck Studio provides a centralized workflow for AI-assisted content generation and multi-platform publishing.

1. Create content
Users generate or prepare original media content inside Nameck Studio.
2. Select a destination platform
Users choose where they want to publish the content.
3. Connect TikTok
Users sign in with TikTok through TikTok Login Kit and explicitly authorize the requested permissions.
4. Choose a video to publish
Users select a video hosted in Nameck Studio for export to TikTok.
5. Review posting settings
Before publishing to TikTok, users review the post settings associated with their TikTok account.
6. Upload to TikTok
Nameck Studio sends the selected video to TikTok using TikTok’s Content Posting API.
7. Finish on TikTok or publish directly
Depending on the enabled TikTok scope and configuration, the video is either sent as a draft for further editing in TikTok or published to the user’s TikTok profile.

Permissions used

user.info.basic: used to identify the connected TikTok account and display the account information in the user flow.

video.upload: used to send selected videos from Nameck Studio to TikTok as drafts.

video.publish: used only when direct posting to TikTok is enabled for the user flow.

Review explanation

Nameck Studio uses TikTok Login Kit to let users securely authenticate with their TikTok account and authorize the required permissions.

Nameck Studio uses TikTok Content Posting API to let users select a video created or managed on the platform and send it to TikTok for publishing.

The user explicitly initiates the export flow from the website and reviews the posting options before the content is transferred to TikTok.
